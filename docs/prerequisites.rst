Prerequisites
*************
.. hlist::
   :columns: 1

   * Install and configure your Switcher device.

   * Collect the following information from the device's following NightRang3r instructions in the
     `Switcher-V2-Python`_ repository:

      * ip_address
      * device_id

.. _Switcher-V2-Python: https://github.com/NightRang3r/Switcher-V2-Python
